네이버웍스 SMTP 메일 발송 설정

help@polarad.co.kr 도메인 메일로 브랜드 리포트 발송하기

왜 변경하나요?

현재 (Gmail API)
  • - "gmail.com을 통해 발송됨" 표시
  • - SPF 불일치 → 스팸 위험
  • - OAuth2 복잡한 설정
변경 후 (네이버웍스 SMTP)
  • - 진짜 help@polarad.co.kr 표시
  • - SPF/DKIM 자동 통과
  • - 계정+비밀번호만 있으면 끝

SMTP 서버 정보

Host: smtp.worksmobile.com
Port: 587
Encryption: STARTTLS
Auth: help@polarad.co.kr + 외부 앱 비밀번호
1

관리자: SMTP 사용 권한 활성화

네이버웍스 Admin 접속
admin.worksmobile.com → 로그인
보안 > 서비스 권한 이동
좌측 메뉴 → 보안 → 서비스 권한
기본 권한 > 구성원 > 권한 수정
메일 항목에서 IMAP/SMTP 체크 → 저장
참고: 이미 활성화되어 있으면 이 단계를 건너뛰세요.
2

계정 생성: help@polarad.co.kr

Admin > 구성원 > 구성원 추가
이름: 폴라애드 고객지원 / 아이디: help
비밀번호 설정
해당 계정의 초기 비밀번호 설정
이미 계정이 있으면 이 단계를 건너뛰고 Step 3로 이동하세요.
3

사용자: 외부 앱 비밀번호 생성

help@polarad.co.kr 계정으로 네이버웍스 로그인
mail.worksmobile.com 접속
우측 상단 프로필 > 설정 > 보안
또는: 설정 > 보안 > 외부 앱 비밀번호
"비밀번호 생성" 클릭
앱 이름: polarad-report 입력 → 생성
생성된 비밀번호 복사
16자리 비밀번호가 표시됨 → 반드시 복사해서 보관
중요: 이 비밀번호는 한 번만 표시됩니다. 닫으면 다시 볼 수 없으니 반드시 복사하세요.
4

나에게 알려줄 것

A
발송 이메일: help@polarad.co.kr (또는 mkt@polarad.co.kr)
B
외부 앱 비밀번호: Step 3에서 생성한 16자리 비밀번호
이 두 가지만 알려주시면, 코드 변경 + 환경변수 설정 + 배포까지 제가 바로 처리합니다.
