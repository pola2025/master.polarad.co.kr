Meta 리드 필드를 아래 JSON의 각 값에 매핑하세요. {{1.xxx}} 형식으로 Meta 모듈의 필드를 연결합니다. 

{
  "name": "{{1.full_name}}",
  "phone": "{{1.phone_number}}",
  "company": "{{1.company_name}}",
  "industry": "{{1.industry}}",
  "adName": "{{1.ad_name}}",
  "secret": "9a413058bd25f57111d00f26e0fe7a72"
}

Meta Lead 필드 매핑 안내

JSON 키 설명 필수
name 리드 이름 (full_name) 권장
phone 전화번호 (+82 형식 OK) 필수
company 회사명 선택
industry 업종 선택
adName 광고 이름 선택
secret 인증 키 (변경 금지) 필수

동작 흐름

  1. Meta Lead Ad 접수
  2. Make가 HTTP Request로 webhook 호출
  3. Airtable에 리드 저장
  4. SMS 즉시 발송
  5. 텔레그램 알림 즉시 발송

* 기존 5분 cron은 백업으로 유지 (webhook 누락 시 보완)
