Upstash Redis 설정 가이드

Vercel Marketplace 경유 · 2026-03 기준 · OTP + Rate Limiting 용

1

Vercel Marketplace에서 Upstash Redis 추가

  1. vercel.com/dashboard 접속
  2. 왼쪽 사이드바에서 Storage 클릭
  3. * 만약 Storage 탭이 없으면: 프로젝트 선택 → 상단 탭에서 Storage → Browse Marketplace
  4. Upstash Redis 선택 → Create
  5. 설정:
    • Name: polarad-redis
    • Region: ap-northeast-1 (도쿄 / 한국 최근접)
    • Type: Regional (Global은 유료)
    • Eviction: Disabled
  6. Create 클릭
2

master-polarad 프로젝트에 연결

  1. 생성된 Redis 상세 페이지에서 Connect Project 클릭
  2. 프로젝트: master-polarad-co-kr 선택
  3. Environment: Production 체크 (Development도 체크 권장)
  4. Connect 클릭
  5. 자동으로 추가되는 환경변수 확인:
    • UPSTASH_REDIS_REST_URL
    • UPSTASH_REDIS_REST_TOKEN
3

polasales (polarad) 프로젝트에도 연결

  1. 같은 Redis 상세 페이지에서 다시 Connect Project
  2. 프로젝트: polarad (polasales 프론트엔드) 선택
  3. Connect 클릭
  4. 같은 UPSTASH_REDIS_REST_URL / UPSTASH_REDIS_REST_TOKEN이 이 프로젝트에도 주입됨
4

로컬 개발 환경에 env 가져오기

터미널에서 각 프로젝트 루트에서 실행:

cd F:\master_polarad
vercel env pull .env.local
cd F:\polasales\website
vercel env pull .env.local

* .env.local에 UPSTASH_REDIS_REST_URL과 UPSTASH_REDIS_REST_TOKEN이 추가되어야 함

5

연결 테스트

클로드에게 알려주면 테스트 실행 후 배포합니다.

# 테스트 명령 (클로드가 실행)
node -e "const {Redis}=require('@upstash/redis'); const r=new Redis({url:process.env.UPSTASH_REDIS_REST_URL,token:process.env.UPSTASH_REDIS_REST_TOKEN}); r.set('test','ok').then(()=>r.get('test')).then(v=>console.log('Redis OK:',v)).catch(e=>console.error(e))"

요약

서비스 Upstash Redis (Vercel Marketplace)
리전 ap-northeast-1 (도쿄 / 한국 최근접)
무료 한도 10,000 commands/일, 256MB
용도 OTP 저장 + 로그인 레이트리밋
환경변수 UPSTASH_REDIS_REST_URL + UPSTASH_REDIS_REST_TOKEN
연결 프로젝트 master-polarad-co-kr + polarad
