Search Console API 설정 가이드

유입 검색어 데이터를 대시보드에 연동하기 위한 설정 가이드

사이트 URL: https://polarad.co.kr

사전 요구사항

  • GA4 설정 가이드의 1~3단계 완료 (서비스 계정 생성)
  • Google Search Console에 polarad.co.kr 사이트 등록

진행 체크리스트

1

Search Console API 활성화

  1. 1.1
    Google Search Console API 페이지 접속
  2. 1.2
    GA4 설정 시 생성한 프로젝트(polarad-dashboard)가 선택되어 있는지 확인
  3. 1.3
    사용 버튼 클릭
2

Search Console에 서비스 계정 추가

  1. 2.1
    Google Search Console 접속
  2. 2.2
    왼쪽 메뉴에서 설정 클릭
  3. 2.3
    사용자 및 권한 클릭
  4. 2.4
    사용자 추가 클릭
  5. 2.5
    서비스 계정 이메일 입력 (예: ga4-reader@polarad-dashboard.iam.gserviceaccount.com)
  6. 2.6
    권한: 전체 또는 제한됨 선택 → 추가

권한 설명

  • 전체: 모든 데이터 확인 가능 + 설정 변경 가능
  • 제한됨: 대부분의 데이터 확인 가능 (권장)
3

환경 변수 설정

기존 환경 변수에 다음을 추가합니다:

.env.local (추가) 
# Google Search Console 설정
SEARCH_CONSOLE_SITE_URL=https://polarad.co.kr

전체 환경 변수 예시

# Google Analytics 4 설정
GA4_PROPERTY_ID=514776969

# Google Search Console 설정
SEARCH_CONSOLE_SITE_URL=https://polarad.co.kr

# 서비스 계정 (GA4와 Search Console 공통 사용)
GOOGLE_SERVICE_ACCOUNT_EMAIL=xxx@project.iam.gserviceaccount.com
GOOGLE_PRIVATE_KEY="-----BEGIN PRIVATE KEY-----\n...\n-----END PRIVATE KEY-----"

설정 완료!

모든 단계를 완료하면 대시보드에서 유입 검색어 데이터를 확인할 수 있습니다.
