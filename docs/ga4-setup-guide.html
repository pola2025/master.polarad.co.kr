GA4 Data API 설정 가이드

폴라애드 대시보드에 Google Analytics 4 데이터를 연동하기 위한 설정 가이드

GA4 속성 ID: 514776969

진행 체크리스트

1

Google Cloud Console 프로젝트 생성

  1. 1.1
    Google Cloud Console 접속
  2. 1.2
    상단의 프로젝트 선택 드롭다운 클릭 → 새 프로젝트 클릭
  3. 1.3
    프로젝트 이름 입력:
    polarad-dashboard
  4. 1.4
    만들기 버튼 클릭
2

Google Analytics Data API 활성화

  1. 2.1
    Google Analytics Data API 페이지 접속
  2. 2.2
    방금 생성한 프로젝트(polarad-dashboard)가 선택되어 있는지 확인
  3. 2.3
    사용 버튼 클릭
3

서비스 계정 생성 및 키 다운로드

  1. 3.1
    서비스 계정 페이지 접속
  2. 3.2
    + 서비스 계정 만들기 클릭
  3. 3.3
    서비스 계정 이름 입력:
    ga4-reader
  4. 3.4
    만들고 계속하기 클릭 → 완료 클릭
  5. 3.5
    생성된 서비스 계정 클릭 → 키 탭
  6. 3.6
    키 추가 → 새 키 만들기 → JSON 선택 → 만들기
  7. 3.7
    중요: 다운로드된 JSON 파일을 안전하게 보관하세요. 이 파일은 다시 다운로드할 수 없습니다.
  8. 3.8
    서비스 계정 이메일 복사 (형식: ga4-reader@polarad-dashboard.iam.gserviceaccount.com)
4

GA4 속성에 서비스 계정 추가

  1. 4.1
    GA4 Admin 접속
  2. 4.2
    속성 열 → 속성 액세스 관리 클릭
  3. 4.3
    + 버튼 → 사용자 추가
  4. 4.4
    서비스 계정 이메일 입력 (3.8에서 복사한 이메일)
  5. 4.5
    역할: 뷰어 선택 → 추가
5

환경 변수 설정

다운로드한 JSON 파일에서 다음 값들을 추출하여 환경 변수로 설정합니다:

.env.local 
# Google Analytics 4 설정
GA4_PROPERTY_ID=514776969

# 서비스 계정 (JSON 파일에서 추출)
GOOGLE_SERVICE_ACCOUNT_EMAIL=서비스계정이메일@프로젝트.iam.gserviceaccount.com
GOOGLE_PRIVATE_KEY="-----BEGIN PRIVATE KEY-----\n...\n-----END PRIVATE KEY-----\n"

Vercel 배포 시

  1. 1. Vercel 프로젝트 설정 → Environment Variables
  2. 2. 위 환경 변수들을 추가
  3. 3. GOOGLE_PRIVATE_KEY는 JSON 파일의 private_key 값 그대로 입력 (따옴표 포함)

설정 완료!

모든 단계를 완료하면 대시보드에서 GA4 데이터를 확인할 수 있습니다.
